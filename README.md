# podcast-cutter
Automatic audio file chopper.

## CSV templates

Section 02 "Segments" includes downloadable CSV files for both single-file cuts and batch processing:

- `segments-template.csv` for blank single-file segment imports
- `segments-sample.csv` for a filled single-file example
- `segments-batch-template.csv` for blank batch imports
- `segments-batch-sample.csv` for a filled batch example

Sample rows prefixed with `__sample__` are ignored by the importer, so the example files can be uploaded safely.
